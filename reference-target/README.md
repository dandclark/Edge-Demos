# Reference Target demos

➡️ **[Open the demo](https://microsoftedge.github.io/Demos/reference-target/)** ⬅️

This page shows interactive demos of the [Reference Target](https://github.com/WICG/webcomponents/blob/gh-pages/proposals/reference-target-explainer.md) proposal, which allows ID references to cross shadow DOM boundaries using the `referenceTarget` attribute on shadow roots.

## Try the feature

To try the feature, enable the **Experimental Web Platform features** flag at `about://flags` in Microsoft Edge or another Chromium-based browser.
